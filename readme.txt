Accept AstroPay payment methods in your WooCommerce store.

== Description ==
This plugin adds AstroPay Payment Gateway for WooCommerce to allow customers access to multiple local payment methods

= Usage =
Just create an account with us in [our site](https://merchant.astropaycard.com/sign_up), add the plugin to your WooCommerce store, enable it, configure your API Credentials, and you are ready to start processing payments with us!
Once the customers select our payment methods, they will be redirected to our Hosted Checkout where they will see the details needed to complete the payment and get back to your website.

= Features =  
Show your customers with the several payment methods we have on their countries updated real-time.
Create an order with only the minimum amount of information and automatically get notificated when those are paid by the customers!

== Installation ==
1. You can:
 * Upload the `astropay-wc-gateway` folder to `/wp-content/plugins/` directory via FTP. 
 * Upload the full ZIP file via *Plugins -> Add New -> Upload* on your WordPress Administration Panel.
 * Search **AstroPay for WooCommerce** in the search engine available on *Plugins -> Add New* and press *Install Now* button.
2. Activate plugin through *Plugins* menu on WordPress Administration Panel.
3. Set up plugin on *WooCommerce -> Settings -> Payments -> AstroPay*.
4. For obtaining credentials, please register at [AstroPay](https://merchant.astropaycard.com/sign_up)
